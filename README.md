# 技术分享 -- C 栈帧和 UNIX 进程切换

node 版本大于等于 v8.4.0

```
npm install
或者
npm install chalk

node main.js
```

## 目录

1. 封面 关于 banner 命令
2. 引子 chmod 777 命令开启系统大门
3. 栈帧 -- 什么是函数？
4. 一个例子：编写一个 tiny 操作系统用来跑一个函数
5. 记不住的系统调用fork
6. 一个例子：修改栈帧用来模拟进程切换
7. 真实的UNIX的进程切换
7. 虚拟内存（如果有时间可能会讲讲）
8. 进程 image 和一张图看清内存（如果做出来一个web再讲）

新建进程流程：
    1. newproc 生成新进程
    2. sched 开始调度
    3. sleep 当前进程等待
    4. swtch 切换进程 （此处修改了栈帧和返回值）


普通分时逻辑
    1. 1/20s 间隔发生时钟中断
    2. psw 和 pc 压入内核栈
    3. 中断处理函数 call 和 clock
    4. 保存 r0 r1, lions 570行 779行
    5. sched 开始调度
    6. sleep 当前进程等待
    7. swtch 切换进程 （此处修改了栈帧和返回值）
    8. 返回中断处理函数 call 和 clock
    9. 恢复 r1 r0 （返回值被覆盖） lions 802行 804行
